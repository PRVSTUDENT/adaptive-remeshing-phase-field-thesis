\section{Workflow}
Auxiliary continuum analysis $\to$ MISESERI extraction $\to$
normalized marking (\texttt{relative\_MISESERI\_threshold}$=0.05$) $\to$
localized graded mesh $\to$ U1/U2/CPS4 reconstruction $\to$
deterministic validation $\to$ four-thread UEL solution $\to$
comparison against uniform H1.

\section{Failed iteration and correction}
Missing exact $y=0$ prevented notch free-face reconstruction (continuous plate;
stiffness $+72\%$). After restoring $y=0$ and doubled notch faces, the elastic
probe error was $0.083\%$ and C2F-v3 recovered H1 peak/pre-peak response.

\section{Non-claims}
This is \emph{not} native Abaqus adaptive remeshing; full-curve and crack-path
mesh independence are not claimed; walltime must not be attributed solely to
refinement without a fair four-thread H1 baseline.
