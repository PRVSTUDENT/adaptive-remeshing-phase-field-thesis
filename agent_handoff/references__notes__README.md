# Literature Notes

This folder contains structured extraction notes rather than thesis prose. Each note records the source role, equations and conventions to extract, implementation details, benchmark details, and thesis decisions affected by the paper.

Use these files with:

```powershell
python scripts/validation/check_literature_index.py --index references/notes/literature_index.json
```

The rapid study guide is a checklist and synthesis only; original papers remain the source of record for equations, benchmark parameters, and implementation claims.
